# appjsreport

-> Sistema de geração de pdf utilizando [handlebars](http://handlebarsjs.com/) e phantom pdf

## Tecnologias:
* Front-end: HTML, JS, CSS, angularjs
* Back-end: Nodejs, Express, Phantom-pdf
* DB: MongoDB

## Run this project

> npm install

> sudo service mongod start

> node server

>> Open http://localhost:8000
